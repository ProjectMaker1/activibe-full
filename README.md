# ActiVibe – Full Stack Starter (Web + PWA)

This monorepo contains:

- `frontend/` – React + Vite PWA
- `backend/` – Node.js + Express + Prisma + PostgreSQL
- `shared/` – Shared business logic / utilities (planned to be reused in future React Native app)
